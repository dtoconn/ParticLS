class particls\+::\+Peridynamic\+Force\+State$<$ dim $>$}

A force state for a peridynamic model. 

Note\+: this is a peridynamic force state, which defines the force per volume squared for a peridynamic model. This is N\+OT a \mbox{\hyperlink{classparticls_1_1_state_object}{State\+Object}} because it does not need to be integrated in time.
