\doxysubsection{Detailed Description}
Forward declaration of rigid body. 

Defines rigid bodies with translation and rotation.

Defines rigid body particles and adds orientation and angular velocity states to the \mbox{\hyperlink{classparticls_1_1_particle}{Particle}} class. The idea is that the particle class could be used for more general meshless methods (like peridynamics) that do not require rotations, whereas the \mbox{\hyperlink{classparticls_1_1_rigid_body}{Rigid\+Body}} class is more specific to the discrete element method.
