This file contains a basic prescription for performing string registration. It allows you to go from a string and get a function pointer something that you can call like make\+\_\+shared$<$\+T$>$.

If you\textquotesingle{}re trying to register a class, and were directed here, then add\+: R\+E\+G\+I\+S\+T\+E\+R\+\_\+\+Y\+O\+U\+R\+C\+L\+A\+S\+S\+N\+A\+M\+E(\+N\+A\+M\+E) where Y\+O\+U\+R\+C\+L\+A\+S\+S\+N\+A\+ME is the type of registration, and N\+A\+ME is the unqualified name of the class to register.
