\doxysubsection{Detailed Description}
\subsubsection*{template$<$unsigned int dim$>$\newline
class particls\+::\+V\+T\+K\+Data$<$ dim $>$}

Collects data to be written to .vtp file. 

This class contains all the methods and variables used for visualizing particles. V\+TK is used to write files to be loaded into Para\+View. An object of this class should be created every timestep (often in Collection.\+cpp). Different particles are added to the class in their respective Geometry.\+cpp file. Specific coloring options are specified in a vector of Save\+Data options in Simulation.\+cpp. Each of the specified options will be added to the geometry in a switch statement in Rigid\+Body.\+cpp.
