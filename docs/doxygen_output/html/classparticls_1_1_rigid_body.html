Detailed Description

Forward declaration of rigid body.

Defines rigid bodies with translation and rotation.

Defines rigid body particles and adds orientation and angular velocity states to the Particle class. The idea is that the particle class could be used for more general meshless methods (like peridynamics) that do not require rotations, whereas the RigidBody class is more specific to the discrete element method.

virtual Eigen::Matrix<double, dim, 1> particls::RigidBody::ProximalOperator ( Eigen::Ref< const Eigen::Matrix< double, dim, 1 >> const &  x,
Eigen::Ref< const Eigen::Matrix< double, dim, 1 >> const &  ref_pt,
double  lambda 
) const
virtual

For a function \(f(x)\), the proximal operator with weight \(\lambda\) at a point \(x\) is defined as

\[ \text{prox}_{\lambda f} = \argmin_y\left(f(y) + \frac{1}{2\lambda}|x-y|^2_2 \right). \]

This operator arises in many optimization algorithms and is key component of ParticLS's optimization-based contact detection routines. This function applies the proximal operator for the level set function of the particle. It is used in RigidBody::ConvexComputeContact and calls the Geometry::ProximalOperator.

Parameters
[in]xThe point \(x\) where we want to evaluate the proximal operator.
[in]ref_ptThe reference point used for one-side SDF's line the flatboundary
[in]lambdaA double containing the weight \(\lambda\).
Returns
The proximal operator for the level set function applied to the point \(x\).
