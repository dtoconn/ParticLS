◆ GetNodes()

template<unsigned int dim>
std::vector<unsigned int> particls::RegularGrid< dim >::GetNodes ( unsigned int  cell_ind) const

Returns the linear indices of nodes surrounding a cell. Notice that for a single cell and a particular coordinate direction, there are only two possible locations for a node, e.g., the left and right of the cell for the x direction. Thus, there are 2^2 nodes in 2D and 2^3 nodes in 3D. The nodes are returned so that the x direction changes the most quickly, followed by the y direction, and in 3d, the z direction. For example, if [x1,x2], [y1,y2], [z1,z2] are the possible values in each coordinate direction, then this function would return the linear node indices corresponding to these locations:

  • In 2D: (x1,y1), (x2, y1), (x1, y2), (x2,y2)
  • In 3D: (x1, y1, z1), (x2, y1, z1), (x1, y2, z1), (x2, y2, z1), (x1, y1, z2), (x2, y1, z2), (x1, y2, z2), (x2, y2, z2)
