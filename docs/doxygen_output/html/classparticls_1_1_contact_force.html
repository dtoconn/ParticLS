◆ Removeable()

template<unsigned int dim>
virtual bool particls::ContactForce< dim >::Removeable ( ) const
virtual

Can this contact for be removed?

Some contact forces should not be removed even if the two particles are far apart. For example, existing cohesive beams should not be removed until after they break. Defaults to returning true always. Can be overriden by children.

Returns
True: the contact force between two particles can be removed, False: the contact force between two particles cannot be removed.

Reimplemented in particls::PeridynamicForce< dim >, and particls::CohesiveBeam< dim >.
