Detailed Description

template<unsigned int dim>
class particls::PeridynamicForceState< dim >

A force state for a peridynamic model.

Note: this is a peridynamic force state, which defines the force per volume squared for a peridynamic model. This is NOT a StateObject because it does not need to be integrated in time.
