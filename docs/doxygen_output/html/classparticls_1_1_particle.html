Detailed Description

template<unsigned int dim>
class particls::Particle< dim >

A particle for a discrete element model.

Defines the particle's position, velocity, angle of rotation (2D)/quaternion (3d), and angular velocity. Also includes any other state variables associated with the particle (e.g., temperature)

◆ AddForce() [1/2]

template<unsigned int dim>
virtual bool particls::Particle< dim >::AddForce ( std::shared_ptr< GenericForce< dim >> const &  new_force)
virtual

Adds a new force to this particle. If the force is a ContactForce, it is added to the pair_forces map. Otherwise, it is added to the generic_forces vector. If the new force is a contact force for involving a particle that this particle is already in contact with, then the new force is rejected and this function returns false. Otherwise, the new force is added to the map of contact forces and this function returns true. For non contact forces, this function always returns true.

Parameters
[in]new_forceA new force (contact, body, or generic) to add to this particle
Returns
A boolean indicating if the new force was succesfully added to the particle.

int const  groupnum

virtual

Adds a new force to this particle but only adds it to particles of a specified group. If the force is a ContactForce then the force is added to the pair_forces map regardless of the group number. If the force is a ContactForce, it is added to the pair_forces map. Otherwise, it is added to the generic_forces vector. If the new force is a contact force involving a particle that this particle is already in contact with, then the new force is rejected and this function returns false. Otherwise, the new force is added to the map of contact forces and this function returns true. For non-contact forces, this function always returns true.

Parameters
[in]new_forceA new force (contact, body, or generic) to add to this particle
[in]groupnumThe group to add this force too
Returns
A boolean indicating if the new force was succesfully added to the particle.
