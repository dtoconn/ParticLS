Detailed Description

template<unsigned int dim>
class particls::CohesiveBeam< dim >

Cohesive beam model for adhesive contact.

The cohesive beam model constructs a linear elastic Euler-Bernoulli beam between particle pairs to approximate the behavior of a continuous material.
