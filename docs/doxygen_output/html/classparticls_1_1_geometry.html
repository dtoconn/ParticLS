Return a bounding box for this geometry in global coordinates defined by the position and orientation arguments. The first row of the output is a lower bound for each coordinate direction and the second row is an upper bound for each coordinate direction. The position and orientation passed to this function are typically taken from a specific particle and define the translation and rotation of the geometry's reference coordinates.

Parameters

The curvature is defined as

\[ \kappa = \nabla \cdot ( \frac{\nabla \phi}{\|\nabla \phi\|_2} ) \]

for signed distance function \(\phi(x)\)
