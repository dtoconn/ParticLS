Observer that tracks a collection's state throughout a simulation. The default state is the collection's homogenized Cauchy stress tensor, but it can also return the collection's 'KineticEnergy' or 'PotentialEnergy'.

template<unsigned int dim>
virtual Eigen::VectorXd particls::CollectionTracker< dim >::QuantityOfInterest ( double const  time)
overridevirtual

This returns the stress tensor in a row-by-row flattened vector. The flattening is necessary because the QuantityOfInterest() returned by other observers are Eigen::VectorXd's. In order to measure the timeseries for multiple observers in one Simulation->Run(std::vector<Eigen::MatrixXd>& observations) instance, all QuantityOfInterest() functions need to return the same data structure. @seealso Simulation
