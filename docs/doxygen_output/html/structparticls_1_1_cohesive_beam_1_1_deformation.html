Detailed Description

template<unsigned int dim>
struct particls::CohesiveBeam< dim >::Deformation

This object describes the amount of deformation (compression, tension, bending, torsion) on the beam.

We use this to decide when the beam breaks.

Holds: The amount that the beam has been stretched/compressed ( \(\Delta l\)), the amount that the beam as been bent (in 3D, return the maximum bending normal to the beam), and the torsion (3D only)
