The first thing you need to do is install the required dependencies:

  • yaml-cpp We recommend installing yaml-cpp from source using cmake.
    1. git clone --depth=1 https://github.com/jbeder/yaml-cpp
    2. cd yaml-cpp; mkdir build; cd build
    3. @codecmake -DCMAKE_INSTALL_PREFIX=</Some/Install/Path/> -DBUILD_SHARED_LIBS=ON ../
    4. make -j4 install
  • eigen3 Again, we recommend installing eigen3 from source using cmake.
    1. git clone --depth=1 https://github.com/eigenteam/eigen-git-mirror.git
    2. cd eigen-git-mirror; mkdir build; cd build
    3. cmake -DCMAKE_INSTALL_PREFIX=</Some/Install/Path/> ../
    4. make -j4 install
